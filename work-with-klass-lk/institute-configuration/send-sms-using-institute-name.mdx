description: 'අධ්‍යාපන ආයතනයේ නමින් SMS යවන්නේ කෙසේද?'
---

පළමුව admin dashboard එක වෙත පැමිණ එහි ඇති settings tab එක click කරන්න.

![Settings Tab](/images/klass-work/settings-tab.png)

ඉන්පසු පැමිණෙන මුහුණතෙහි Config tab එක click කර එහි පහලට ගොස් Terrareach API Details කොටසට පැමිනෙන්න.

![Config Tab](/images/klass-work/config-tab.png)
![TR API Details](/images/klass-work/tr-api-details.png)

මෙහිදී ඔබට ඔබේ ආයතනයේ නම සහිත Sender ID එකක් සහ API key එකක් ඇතුලත් කිරීමට සිදුවේ. ඒ සඳහා පහත පරිදි link එක කරන්න.

![TR Link](/images/klass-work/go-to-tr-link.png)

එහිදී ඔබට [terrareach.com](https://terrareach.com) හි ලියාපදිංචි වීමට සිදුවේ. අනතුරුව පහත පරිදි මුහුණතක් දිස්වේ. එහි ඔබ ලියාපදිංචි වූ නම සහිත dropdown එක click කර settings වෙත යන්න.

![TR Settings](/images/klass-work/tr-settings.png)

එහි Sender IDs tab එක click කර ඔබේ ආයතනයට SMS යැවීමට අවශ්‍ය Sender ID එක ඉල්ලුම් කරන්න.

![TR Sender IDs](/images/klass-work/tr-sender-ids.png)

අනතුරුව API Keys tab එක click කර API Key එකක් සාදා ගන්න.

![TR API Keys](/images/klass-work/tr-api-keys.png)

මෙම icon එක click කිරීමෙන් ඔබට සාදාගත් API Key එක copy කර ගත හැකිය.

![TR API Key Copy](/images/klass-work/tr-api-copy.png)

ඉන්පසු නැවත klass.lk වෙත පැමිණ එහි ඇති මෙම Edit Button එක click කරන්න.

![Add TR API](/images/klass-work/add-tr-api.png)

ඉන්පසු TerraReach Sender ID හි ඔබ ඉල්ලුම් කරන ලද Sender ID එක නිවැරදිව ඒ ආකාරයෙන්ම ඇතුලත් කර TerraReach API Key හි ඔබ සාදා ගත් API Key එක ඇතුලත් කර Save Changes Button එක click කරන්න

![Save TR API](/images/klass-work/save-tr-api.png)

දැන් ඔබ සාර්ථකව ඔබේ අධ්‍යාපන ආයතනයේ නමින් SMS යැවීමට සියල්ල සූදානම් කරගෙන අවසන්. ඔබේ ආයතනයට සිසුන් ලියාපදිංචි වීමේදී, මුදල් ගෙවීමේදී SMS මගින් පණිවිඩය සිසුන්ට ලැබේ.
