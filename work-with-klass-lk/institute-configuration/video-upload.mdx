description: 'Video upload කිරීම පිළිබඳ සැකසුම් සාදා ගන්නේ කෙසේද?'
---

පළමුව admin dashboard එක වෙත පැමිණ එහි ඇති settings tab එක click කරන්න.

![Settings Tab](/images/klass-work/settings-tab.png)

ඉන්පසු පැමිණෙන මුහුණතෙහි Config tab එක click කර එහි ඔබගේ වීඩියෝ පාඩම් ගබඩා කරන Vimeo video hosting platform එකෙහි ඔබගේ ගිණුමට අදාල API key එක ඇතුලත් කරන්න.

![Video upload](/images/klass-work/video-upload.png)
