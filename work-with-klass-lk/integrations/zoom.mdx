## Zoom එකතු කිරීම

පළමුව admin dashboard එක වෙත පැමිණ එහි ඇති integrations tab එක click කරන්න.

![Integration Tab](/images/klass-work/klass-dashboard.png)

ඉන්පසු පැමිණෙන මුහුණතෙහි ඇති Zoom card එකෙහි ඇති Connect button එක click කරන්න.

![Zoom Connect Button](/images/klass-work/zoom-connect-button.png)

ඉන්පසු ඔබව Zoom Login page එක වෙත ගෙන යයි. එහි ඔබේ Zoom ගිණුමේ Email සහ Password ලබා දී login වෙන්න.

![Zoom Login Page](/images/klass-work/zoom-login.png)

ඉන් පසුව ඔබේ Zoom account එකට access ලබා ගන්​න klass.lk වෙත permission ලබා දීමට එකඟවන page එක දිස්වේ. එහි පහල ඇති Allow button එක click කර permission ලබා දෙන්න. 

![Zoom Permission Page](/images/klass-work/zoom-permission-allow.png)

ඉන්පසුව නැවත ඔබව Klass.lk integrations tab එක වෙත පැමිණේ. දැන් එහි තිබූ Connect button එක කොල පාටින් Connected ලෙස දිස්වේ. මෙයින් අදහස් වන්නේ ඔබ සාර්ථකව Zoom integration එක කරගෙන ඇති බවයි.

![Zoom Connected](/images/klass-work/zoom-connected.png)

## Zoom භාවිතා කිරීම

දැන් ඔබට පහත පියවර අනුගමනය කර නැවත Zoom වෙත නොයා Klass.lk platform එක තුලින්ම Zoom session සාදා ගන්න.

Course tab එක වෙත පැමිණ එහි ඇති courses list එකේ ඇති module button එක click කරන්න.

![Zoom Course List](/images/klass-work/zoom-course-list.png)

ඉන්පසු පැමිණෙන module list එකේ ඇති manage session button එක click කරන්න.

![Zoom Manage Sessions](/images/klass-work/zoom-manage-sessions.png)

එහි ඇති Add Session button එක click කර Zoom session එකක් add කරන්න.

![Zoom Add Session](/images/klass-work/zoom-add-session.png)

එහිදී එන modal එකෙහි Resource Type එක යටතේ ඇති Zoom Meeting option එක මේ සඳහා තෝරාගත යුතුය.

![Zoom Resource Type](/images/klass-work/zoom-resource-type.png)
![Zoom Meeting](/images/klass-work/zoom-meeting.png)

එහි ඇති ඉතිරි තොරතුරු එකතු කර Save button එක click කර Zoom session එක එකතු කරගන්න.

![Zoom Session Modal](/images/klass-work/zoom-session-modal.png)

ඉන්පසු ඔබට එම Zoom session එක මෙලෙස දිස්​වේ. එහි ඇති Video icon එක click කල විට Zoom session එකෙහි විස්තර දිස්වේ.

![Zoom Session List](/images/klass-work/zoom-session-list.png)

එහි ඇති Start Meeting Button එක click කිරීමෙන් ඔබට session එක ආරම්භ කළ හැකිය.

![Zoom Session Details](/images/klass-work/zoom-session-details.png)


## Zoom ඉවත් කිරීම

කොල පාටින් දිස් වෙන Connected button එක click කරන්න.

![Zoom Connected](/images/klass-work/zoom-connected.png)

එවිට ඔබට මෙය ඉවත් කිරීම තහවුරු කිරීමට පහත පරිදි modal එකක් දිස්වේ.

![Zoom Disconnect](/images/klass-work/zoom-disconnect.png)

එහි Yes button එක click කල විට ඔබට Zoom integration එක ඉවත් වේ.
